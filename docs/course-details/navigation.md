# Navigating this course

## How to use this site

- The **top tabs** group the course into sections: Course details, Climate
  datasets, Exercises, Scripts & code, and Resources.
- The **left sidebar** lists the pages in the current section.
- The **right sidebar** is the on-page table of contents.
- Use the :material-magnify: **search** box (top) to jump to any topic, dataset
  ID or country.
- Toggle :material-weather-night: **dark mode** with the icon in the header.

## Suggested path

1. [Set up your account](../exercises/account-setup.md) — do this first.
2. [Code Editor basics](../exercises/code-editor.md) — find your way around.
3. Work through exercises **3 → 5** in order; each builds on the last.
4. Dip into [Google Earth Engine](../datasets/what-is-gee.md) and the
   [Glossary](../resources/glossary.md) whenever you want background.

## Copying code

Every code block has a :material-content-copy: **copy button** in its top-right
corner. Copy the block, paste it into the Earth Engine Code Editor, and click
**Run**. The full scripts live under [Scripts & code](../scripts/index.md) and
can also be downloaded.

## Localising for your country

Wherever a script has a line like `var COUNTRY = 'Fiji';`, change it to your
nation. See [Code Editor basics](../exercises/code-editor.md) for the important
detail about country names and small islands.
